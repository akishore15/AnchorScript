About AnchorScript

AnchorScript is a custom programming language designed to make coding simple and fun. It uses unique syntax such as div before any data type and var() for variable declarations.

Getting Started

To get started with AnchorScript, you can download the interpreter and start writing your own programs. Here's a simple example:

div int a = 5;
div int b = 10;
div int c = var(a) + var(b) * 2;
div float d = 2.5;
div bool e = var(c) > var(b);
print;
print;
print;
